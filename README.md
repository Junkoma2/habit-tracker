# 習慣トラッカー

毎日の習慣を記録するシンプルな Web アプリです。

**→ https://junkoma2.github.io/habit-tracker/**

## 機能

- 習慣の追加・編集・削除・並び替え（ドラッグ操作）
- 今日・昨日の記録（長押しで切り替え）
- カレンダーで達成状況を確認
- 統計表示（連続日数・最長連続・累計達成回数）
- pull-to-refresh で最新状態に更新
- テーマカラーの変更（6色プリセット）
- データのバックアップ・復元（設定 → データ管理、JSON形式）
- ブラウザのローカルストレージに自動保存（外部送信なし）

## 使用技術

- React / Vite
- @dnd-kit（ドラッグ＆ドロップ）
- GitHub Pages
